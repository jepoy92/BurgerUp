
# BurgerUp
[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)
## Description 

A burger logger with MySQL, Node, Express, Handlebars and a homemade ORM (yum!) that uses Node and MySQL to query and route data in your app, and Handlebars to generate your HTML.

## Installation

Simply type "npm i" in your command line to install all dependencies

## Usage 

Log the burgers you've encountered or make up your own to keep track of what you've liked or disliked. Click on yum to keep the burgers you liked and delete to toss the burgers you didn't1
